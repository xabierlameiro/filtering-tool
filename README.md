# Filtering Tool

A Next.js application demonstrating dynamic filtering and search UI patterns
with TypeScript, React, and the App Router.

[![CI](https://github.com/xabierlameiro/filtering-tool/actions/workflows/ci.yml/badge.svg)](https://github.com/xabierlameiro/filtering-tool/actions/workflows/ci.yml)

## Stack

| Layer     | Choice               |
| --------- | -------------------- |
| Framework | Next.js (App Router) |
| Language  | TypeScript           |
| Library   | React                |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
