## ¿Cómo crear un componente controlado en React?

# epa

Un componente controlado en React es aquel cuyo estado es manejado completamente por React y se actualiza a través de sus props. ¿Puedes escribir un ejemplo de un componente controlado en React utilizando el hook `useState`?

```tsx {2}
<Fragment>
  <input type="text" value={value} onChange={handleChange} />
</Fragment>
```
